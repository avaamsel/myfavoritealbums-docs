Reference Documents for Developers

API Reference:

albums_by_band.R:

The functions in the albums_by_band.R file create the functionality in MyFavoriteAlbums’ Artists’ Albums tab.

albums_by_year.R:

The function in the albums_by_year.R file creates the functionality in MyFavoriteAlbums’ Top Albums by Year tab.

compare_bands.R:

The function in the compare_bands.R file creates the functionality in MyFavoriteAlbums’ Artist Comparison tab.

fav_bands.R:

The function in the fav_bands.R file creates the functionality in MyFavoriteAlbums’ Favorite Artists tab.

home.R:

The functions in the home.R file create the functionality in MyFavoriteAlbums’ home page.

number_one_albums.R:

The functions in the number_one_albums.R file create the functionality in MyFavoriteAlbums’ Number One Albums tab.

vinyl.R:

The functions in the vinyl.R file create the functionality in MyFavoriteAlbums’ Vinyl tab.

API Reference

albums_by_band.R:

The functions in the albums_by_band.R file create the functionality in MyFavoriteAlbums’ Artists’ Albums tab.

Fuction Name Function Description Function Parameters Function Outputs
albums_by_bands Filters album_data to return all of the rated albums by a specific artist band.var: Specified artist to filter by, chosen in the dropdown band_albums: Dataframe containing the three columns, “Album”, “Year”, “Rating”, which specifies the albums of the selected artist
band_mean_rating Calculates the average rating for the specified artist rounded to the nearest two decimal places band.var: Specified artist to calculate the average rating for, chosen in the dropdown avg_rating: Printed string detailing the average rating for the specified artist
band_album_count Counts the total number of albums ranked for the specified artist band.var: Specified artist to count the number of ranked albums for band_count: Printed string detailing the total number of ranked albums for the specified artists

albums_by_year.R:

The function in the albums_by_year.R file creates the functionality in MyFavoriteAlbums’ Top Albums by Year tab.

Fuction Name Function Description Function Parameters Function Outputs
year_albums Lists all of the top albums for a specified year in ranking order year.var: Specified year to filter the user’s albums by year_list: Dataframe containing the three columns, “Ranking”, “Album”, “Artists”, which specifies all of the albums ranked for the given year, sorted in ranking order

compare_bands.R:

The function in the compare_bands.R file creates the functionality in MyFavoriteAlbums’ Artist Comparison tab.

Fuction Name Function Description Function Parameters Function Outputs
band_album_comparison_chart Compares the ratings of two artists' albums over time on a line chart var.artist1: Name of the first artist to compare

var.artist2: Name of the second artist to comparevar.artist2: Name of the second artist to compare 		ggplot(): ggplot2 line chart comparing the ratings over time for two specified artists

fav_bands.R:

The function in the fav_bands.R file creates the functionality in MyFavoriteAlbums’ Favorite Artists tab.

Fuction Name Function Description Function Parameters Function Outputs
favorite_bands Finds the top 15 bands with at least a specified number of ranked albums, ordered by highest average rating min_albums.var: Number specifying the minimum number of albums an artist must have to be included

live_ep.var: Boolean specifying if the Exclude EPs and Live Albums checkbox has been clicked. If true, EPs and Live albums are not included. If false, EPs and Live albums are included 		fav_artists: Dataframe containing the three columns, “Artist”, “Rating”, “NumRatings”, which specifies the top 15 bands that have the specified number of albums rated

home.R:

The functions in the home.R file create the functionality in MyFavoriteAlbums’ home page.

Fuction Name Function Description Function Parameters Function Outputs
total_album_count Finds the total number of albums in the data set. Subtracts 1 for the row with the headings N/A total_albums: Printed string detailing the total number of albums that are in the data set
total_band_count Finds the total number of bands in the data set. Uses unique to avoid counting the same band twice N/A top_bands: Printed string detailing the total number of bands that are in the data set
most_pop_band Finds the artist in the data set with the most albums N/A popular_artists: Printed string detailing the artist with the most albums in the data set

number_one_albums.R:

The functions in the number_one_albums.R file create the functionality in MyFavoriteAlbums’ Number One Albums tab.

Fuction Name Function Description Function Parameters Function Outputs
number_one_album Lists the number one album for each year in the indicated range var.startyear: Start year of the date range to filter albums from

var.endyear: End year of the date range to filter albums from 		top_albums: dataframe containing the three columns, “Year”, “Album”, “Artist”, which stores the number one ranked album for each year in the specified year range

vinyl.R:

The functions in the vinyl.R file create the functionality in MyFavoriteAlbums’ Vinyl tab.

Fuction Name Function Description Function Parameters Function Outputs
missing_vinyl Finds top-rated albums that the person does not own on vinyl missing_vinyl_rating.var: Number specifying the minimum rating to allow an album to be output buy_vinyl: dataframe containing the three columns, “Album”, “Artist”, “Rating”, which specifies the albums that meet the minimum rating threshold and are not owned on vinyl

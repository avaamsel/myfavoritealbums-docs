Task Topics for Developers:

Setting up the MyFavoriteAlbums Tech Stack:

Cloning the MyFavoriteAlbums Repository:

  1. Access the RStudio terminal by clicking on the Terminal tab at the top of the bottom window.
  2. In your terminal, verify that you are currently located in your Desktop directory. Your directory path should look something like this: [device-name]:Desktop [user]$
  3. If you are not on your desktop, change your directory entering the following command into your terminal: cd Desktop
  4. Clone the MyFavoriteAlbums repository onto your device. git clone https://github.com/UW-Example-Student/MyFavoriteAlbums.git
  5. Verify that the MyFavoriteAlbums repository was correctly cloned onto your desktop by using the ls command. You should see the repository name, MyFavoriteAlbums, output in the list of your desktop contents if it has been cloned correctly. ls Desktop
  6. To download MyFavoriteAlbums’ required packages run the following command in your RStudio console: install.packages(c("shiny", "dplyr", "tidyverse"))

Running MyFavoriteAlbums:

  1. Within the right Files subwindow, navigate to the app.R file.
  2. In the main coding window, click on the Run App button in this window’s navigation bar to open a local version of the MyFavoriteAlbums web application.

Adding New Functions to MyFavoriteAlbums:

To add new functionality to MyFavoriteAlbums, you must have a basic understanding of R. Make sure you are in the MyFavoriteAlbums directory in RStudio.
